## Key decisions

- **Embeddings opt-in and cost-gated, not default.** OpenAI `text-embedding-3-small` is the default provider, key handled via Infisical. Paid calls only fire when configured; absent config, the system falls back to keyword + graph rather than erroring. This keeps the brain usable (and free) out of the box.
- **Hybrid over reranker.** Blend pgvector similarity, keyword match, and graph proximity at query time instead of importing a separate reranker model — keeps the dependency surface and the cost model simple while still beating any single signal.
- **Reindex as a first-class verb.** Rather than forcing a re-mount to get embeddings onto old content, `reindex` backfills in place — important because embeddings shipped after the store already had Phase 1 content, and because switching providers shouldn't mean re-ingesting.

## Scope / process notes

- **Search shipped before wiring** (#275 before #276), inverting the Phase 1 "wire before search" recommendation. It worked out because Phase 1 content (mounted files, entities) gave the retrieval layer something to rank, but the cleaner order remains wire-first.
- **Two brain follow-ups deferred past the release** (not blocking v1.4.0): **ROS-145** (brain organizer — guideline + delegated subagent to structure the brain) and **ROS-146** (index-friendly canonical resolution for large brains, a ROS-140 follow-up). Both remain Backlog under the milestone; the umbrella ROS-134 stays open to track them.

## Learnings to harvest

- **"Degrade, don't fail" is the right default for any paid-dependency feature.** Making semantic search optional-by-fallback means a workspace with no embedding key still gets value from the brain — the feature never becomes a wall.
- **Backfill paths are not optional for stores that gain capabilities over time.** Any feature that derives data from existing rows (embeddings here) needs a reindex verb from day one, or early content silently never benefits.
- The adversarial Codex review loop continued to earn its keep across Phase 2 (consistent with the Phase 1 finding) — budget review→fix rounds on retrieval-ranking and wiring correctness.
